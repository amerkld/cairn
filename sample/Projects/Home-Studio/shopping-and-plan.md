# Garage → home studio

Converting the back half of the garage into a proper small-project woodworking space. Light work only — nothing that needs a dust-collection system bigger than a shop vac.

## Layout sketch

- **Back wall** — workbench (6 ft), pegboard for hand tools
- **Right wall** — 2× shelving units for lumber offcuts
- **Corner** — miter saw station with its own extension cord run
- **Center** — assembly area, 4×8 rolling table
